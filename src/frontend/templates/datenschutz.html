Hosting & Infrastruktur

Diese Website wird auf einem privaten virtuellen Server (VDS) betrieben. Beim Aufruf der Seite verarbeitet der Server technisch notwendig die IP-Adresse der anfragenden Verbindung. Diese wird in den Apache-Zugriffslogs in anonymisierter Form gespeichert – das letzte Oktett der IP-Adresse wird dabei auf .0 gesetzt, sodass keine Rückschlüsse auf einzelne Personen möglich sind.

Zusätzlich ist Cloudflare als vorgelagerter DNS im Einsatz. Cloudflare verarbeitet Verbindungsdaten gemäß der eigenen Datenschutzrichtlinie: cloudflare.com/privacypolicy

Verschlüsselung

Die gesamte Kommunikation zwischen deinem Browser und dieser Website erfolgt verschlüsselt über HTTPS (TLS). Der Inhalt der Wiki-Seiten liegt als Markdown auf dem Server und ist nicht öffentlich zugänglich.

Schriften

Diese Website verwendet die Schriftarten Cinzel und EB Garamond. Beide Schriften sind lokal auf dem Server gespeichert und werden direkt von dort geladen. Es findet keine Verbindung zu externen Diensten wie Google Fonts statt.
